Project Overview

Assembly: Endgame is an interactive word-guessing game inspired by Hangman, built with React. Players must guess the hidden programming-related word one letter at a time before running out of lives. The game provides instant visual feedback, tracks progress in real time, and delivers a fun, engaging experience through a clean and responsive interface.

This project was built to strengthen my understanding of React fundamentals, including state management with Hooks, component composition, conditional rendering, and event handling. It challenged me to manage multiple game states, implement the game logic, and update the user interface dynamically based on player interactions.

Through building Assembly: Endgame, I gained hands-on experience creating interactive applications, writing reusable components, and organizing code for maintainability. The project demonstrates my ability to translate game rules into a functional, user-friendly web application while focusing on performance, accessibility, and a polished user experience.
