Tenzies

Tenzies is an interactive dice game built with React where players roll and strategically hold dice until all ten show the same value. The project demonstrates core React concepts, including Hooks, state management, component composition, and dynamic rendering to deliver a smooth and engaging gameplay experience.

Live Demo

Project Overview

Tenzies is a React-based dice game where the objective is to roll until all ten dice display the same value. Players can hold individual dice between rolls, adding a strategic element to the gameplay while creating an engaging and interactive user experience.

This project was built to strengthen my understanding of React fundamentals, including state management with Hooks, component composition, conditional rendering, and working with arrays of objects. It also focuses on writing clean, reusable code and keeping the interface responsive and intuitive.

While developing Tenzies, I gained practical experience managing complex application state, implementing game logic, and updating the UI efficiently based on user interactions. The result is a simple yet polished browser game that demonstrates core frontend development skills.
